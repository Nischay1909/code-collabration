# Code Collaboration Editor

A browser-based code editor that allows users to write HTML, CSS, and JavaScript with live preview, download functionality, and shareable links.
# Features

- Live code execution (HTML, CSS, JS)
- Real-time preview
- Save code using localStorage
- Download code as HTML file
- Share code via URL
